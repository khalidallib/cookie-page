# Cookie Consent Demo Page

A simple cookie consent and settings demo page built with HTML, CSS, and JavaScript. This project simulates a website cookie popup where users can accept cookies or customize what information is stored.

---

# Tech Stack

- HTML5
- CSS3
- JavaScript

---

# Features

- Cookie consent popup
- Custom cookie settings menu
- User preference toggles
- Responsive layout
- Simple and clean UI

---

# How to Install / Run

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/cookie-page.git
```

## 2. Open the Project

Open the project folder using VS Code or any code editor.

## 3. Run the Application

Open `index.html` in your browser.

You can also use the VS Code Live Server extension for easier testing.
